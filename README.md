# Running the app locally

> Clone and run the API first. API repo is [here](https://github.com/eljohn316/jlabs-assessment-api)

### 1. Clone

```bash
git clone https://github.com/eljohn316/jlabs-assessment-web.git
```

### 2. Setup

```bash
cd jlabs-assessment-web
pnpm install
```

### 3. Running

```bash
pnpm dev
```
